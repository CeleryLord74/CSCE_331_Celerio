What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. HTML is a markup language, which means it uses tags to define elements within the document.

Here is an example of an HTML element: 

<p>This is a paragraph.</p>

In this example, <p> is the opening tag, and </p> is the closing tag. The content between the tags is the text that will be displayed on the webpage.

What is CSS?

CSS stands for Cascading Style Sheets. It is a language used to describe the presentation of an HTML document. CSS controls the layout, colors, fonts, and overall visual appearance of a webpage.

Here is an example of a simple CSS rule: 

p {
    color: blue;
    font-size: 16px;
}

In this example, the CSS rule targets all <p> elements and sets the text color to blue and the font size to 16px.

How HTML and CSS Work Together

HTML provides the structure of the webpage, while CSS defines its style. Together, they create visually appealing and functional websites. Here is an example of how HTML and CSS can be used together: 

<!DOCTYPE html>
<html>
<head>
    <style>
        body {
            background-color: lightgray;
        }
        h1 {
            color: blue;
        }
    </style>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
</body>
</html>

In this example, the CSS is embedded within the HTML file using the <style> tag. The CSS styles the <body> and <h1> elements.
